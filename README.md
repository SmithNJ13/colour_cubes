# Colour Cubes! {WIP!!!!!!!!}

## Install Instructions:
- git clone
- cd  Colour_Cubes
- npm install
- npm run dev

## Game Instructions:
- Sync up all the cubes to the same colour, once they're matched you win that round and your time is recorded!
